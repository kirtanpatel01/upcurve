# Upcurve

**Upcurve** is a habit tracking web application that helps users build and
maintain positive routines. It offers a smooth, responsive interface to add,
track, and manage daily habits with visual streak indicators and real-time
syncing. Built with Next.js App Router and Appwrite, Upcurve ensures a fast and
seamless experience across devices.

---
### Live: [https://upcurve-xi.vercel.app](https://upcurve-xi.vercel.app)
---

## Tech Stack

- Next.js 14 (App Router)
- Tailwind CSS + ShadCN UI
- Appwrite (Auth, DB, Realtime)
