# Upcurve

Upcurve is a modern, personal growth and productivity platform designed to help users track habits, manage tasks, and monitor physical activity. Built with a focus on speed, aesthetics, and user experience, Upcurve provides a unified interface for total life management.

## Core Features

- **Personal Dashboard**: Unified view of task progress, habits, and exercise logs.
- **Task Management**: Simple and efficient todo list with priority levels.
- **Habit Tracking**: Build and monitor daily rituals with consistency history.
- **Exercise Logging**: Track workouts, sets, and progress over time.

## Technical Stack

Upcurve is built with a cutting-edge web stack for maximum performance and developer experience:

- **Framework**: [Next.js 16](https://nextjs.org) (React 19, Turbopack)
- **Authentication**: [Better Auth](https://better-auth.com)
- **Database**: [Neon](https://neon.tech) (Serverless Postgres)
- **ORM**: [Drizzle ORM](https://orm.drizzle.team)
- **State Management**: [Zustand](https://github.com/pmndrs/zustand)
- **Data Fetching**: [TanStack Query v5](https://tanstack.com/query/latest)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com), [DaisyUI](https://daisyui.com)
- **Components**: [Radix UI](https://www.radix-ui.com), [shadcn/ui](https://ui.shadcn.com)
- **Animations**: [Motion](https://motion.dev) (Framer Motion)
- **Charts**: [Recharts](https://recharts.org)
